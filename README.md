### Setup
- Run git clone `https://github.com/tingamapuro04/hello-rails-react.git`
- Run `cd hello_rails_react`
- Run `bundle install`
- Run `npm i`
- Run `npm run build`(having issues building the app Run`npm install --save-dev webpack webpack-cli`)
- Run `rails server`
